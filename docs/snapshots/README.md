# docs/snapshots/

**Замороженные обзоры на дату создания. Не источник правды по текущему коду.**

Это reverse-engineered справочники: PRD, реестр фич, паспорт проекта,
карта инфраструктуры, аудит безопасности БД, индекс функций и т.п.
Полезны как контекст и стартовая точка, когда нужно «общими мазками
понять, что было». Для актуальных деталей — читать код, актуальные
[plans/](../../plans/), и [CLAUDE.md](../../CLAUDE.md).

Любой файл здесь верен на дату из его `created` / `last_updated` поля и
больше не обновляется. Если нужен новый снимок — создать новый файл с
датой, не править старый.
