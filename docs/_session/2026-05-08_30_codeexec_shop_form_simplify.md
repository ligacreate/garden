## Что сделано

В `views/AdminPanel.jsx`, ShopAdmin форма редактирования товара —
удалены из UI:

1. **Секция «Для товаров со скидкой по промокоду»** целиком:
   - заголовок-метка
   - `Input promo_code`
   - `Input link_url`
   - синий контейнер `bg-blue-50/60`

2. **Поле WhatsApp** (`contact_whatsapp`):
   - убран `Input WhatsApp`
   - двухколоночный `grid grid-cols-2` для Telegram/WhatsApp заменён
     на одиночный `Input Telegram` без сетки.

## Что НЕ тронуто (по инструкции стратега)

- `SHOP_EMPTY_FORM` — `promo_code`, `link_url`, `contact_whatsapp`
  остаются в инициализации.
- `openEdit` — все три поля по-прежнему заполняются из `item.*`.
- `payload` при сохранении — все три поля шлются в API
  (`null` если пусто).
- Валидация `if (!form.price && !form.promo_code.trim())` — оставлена
  как есть. После удаления UI `form.promo_code` всегда `''`, поэтому
  логически правило сводится к «нужна цена», что согласуется со
  звёздочкой `Цена (₽) *` в форме.
- БД, миграции, витрина (`MarketView.jsx`) — не тронуты.
- Карточки LOVELIGA / LIGANOTEBOOKS — живут как есть.
