Ольга утвердила переименование UI-раздела.

## Что переименовать

В `views/AdminPanel.jsx`:

- Tab label «Без автопаузы» → **«Льготы»**.
- Заголовок раздела внутри tab — «Льготы».
- Описание/подзаголовок — «Индивидуальные исключения: бартеры,
  постоянные льготы для конкретных людей. Админы и абитуриенты
  защищены автоматически по роли — их в этом списке быть не должно.»

В модалке «Не паузить автоматически» (карточка профиля) — заголовок
оставить «Льгота» (единственное число, потому что для одного юзера).
Чекбокс — «Дать льготу (не паузить автоматически)».

Внутри двух списков:
- «Постоянная льгота» (бывшее «Всегда бесплатно»)
- «Льгота до даты» (бывшее «Бесплатно до даты»)

## Имя поля в БД оставить как есть

`auto_pause_exempt` / `auto_pause_exempt_until` / `auto_pause_exempt_note`
— технические имена, не меняем. Пользователь их не видит.

В коде/доках можно для ясности использовать слово «exempt» в
комментариях, но в UI везде «льгота».

## Где ещё может прозвучать

- Notify-сообщения после save в модалке: «Льгота включена» /
  «Льгота снята».
- Title тултипа на shield-кнопке в строке users: «Льгота: всегда» /
  «Льгота до 2026-12-31» / «Льготы нет».
