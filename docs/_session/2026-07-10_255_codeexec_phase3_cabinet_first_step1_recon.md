## F. ⚠️ ФЛАГ СТРАТЕГУ — гейт курса расходится с моделью «курс = абитуриенты + менторы»

Настоящий гейт курса ПВЛ — **не** level-каскад `hasAccess`, а [`services/pvlRoleResolver.js:15-29`](../../services/pvlRoleResolver.js):

| garden-роль | → PVL-роль | видит курс? |
|---|---|---|
| applicant | student | ✅ |
| **intern** | **student** (строки 22-23, намеренно, коммент «Стажёр… курс ПВЛ доступен») | ✅ |
| mentor | mentor | ✅ |
| admin | admin | ✅ |
| leader | — (fall-through) | ❌ no_access |
| curator | — | ❌ no_access |

- **Модель Оли:** курс = только **абитуриенты + менторы** (+admin как бог).
- **Расхождение ровно на `intern`:** стажёр сейчас курс видит, по модели — не должен. leader/curator уже исключены (совпадает с моделью).

Это шире и другое; если «курс» Оли = ПВЛ, библиотека — отдельный вопрос.
